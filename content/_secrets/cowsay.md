If you've ever used the cowsay tool, this is quite literally a direct port (uses the same cowfile format too)
### API
the query string takes variables as such
- message: what to say
- eyes: 2 characters, replaces eyes "oo" default
- tongue: 2 characters, replaces the tongue "  " default
- cowfile: the cowfile name to use
